# Статический сайт под специфические задачи.
## **Сборка проекта**
### Локально
1. Поместить в `static_manager/in_data` архивы с определенной структурой.
2. Запустить `converter.py`
3. Переместить папку `static_manager/content` в `dvg_frontend\public`
4. Переместить файл `static_manager\db.json` в `dvg_frontend\public`
5. Запустить react vue сервер локально:
```
cd dvg_frontend
npm i
npm run dev
```
